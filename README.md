AI Agent Based Smart Home Automation System using Python
 Project Overview

This project implements a rule-based AI Agent that simulates a Smart Home Automation System using Python.

The system models a smart home environment where the agent senses environmental conditions such as time of day, temperature, presence of people, and natural light level, and automatically performs appropriate actions.

The AI agent makes intelligent decisions to control home functions such as lights, thermostat, and door locking.
